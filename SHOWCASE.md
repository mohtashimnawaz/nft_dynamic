## 📞 Quick Start

```bash
# Install dependencies
yarn install && cd app && yarn install && cd ..

# Build program
anchor build

# Deploy to devnet
solana config set --url devnet
solana airdrop 2
anchor deploy

# Start frontend
cd app && yarn dev
```

Open `http://localhost:5173` and enjoy! 🎉
